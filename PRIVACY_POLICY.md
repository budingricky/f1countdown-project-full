# F1 倒计时 隐私政策

**最后更新日期：2026年2月16日**

---

## 引言

感谢您使用 F1 倒计时应用（以下简称"本应用"）。我们非常重视您的隐私保护，本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

使用本应用即表示您同意本隐私政策的条款。如果您不同意本隐私政策，请不要使用本应用。

---

## 信息收集

### 我们收集的信息

#### 1. 应用使用数据

本应用**不收集**任何个人身份信息。

#### 2. 用户偏好设置

我们在您的设备本地存储以下偏好设置：

- 通知提醒时间偏好（15/30/60分钟）
- 通知提醒的比赛环节偏好
- 收藏的赛道列表
- Pro 购买状态

**重要说明：** 所有偏好设置仅存储在您的设备本地，不会上传到我们的服务器。

#### 3. 赛程数据

本应用从 Jolpica-F1 公开 API 获取 F1 赛程数据。这些数据包括：

- 比赛日期和时间
- 赛道信息
- 赛季赛程

这些数据属于公开信息，不包含任何个人数据。

### 我们不收集的信息

本应用**明确不收集**以下信息：

- ❌ 姓名、电子邮件地址、电话号码
- ❌ 设备标识符（IDFA、IDFV）
- ❌ 精确地理位置
- ❌ 浏览历史
- ❌ 用户行为分析数据
- ❌ 任何可用于识别您身份的信息

---

## 信息使用

### 数据使用目的

我们使用本地存储的数据仅用于：

1. **提供核心功能**
   - 显示赛程倒计时
   - 发送本地通知提醒
   - 同步偏好设置

2. **改善用户体验**
   - 记住用户偏好
   - 保持 Pro 购买状态

### 第三方数据共享

我们**不与任何第三方共享**您的数据。

---

## 数据存储

### 本地存储

所有用户数据存储在您的设备本地：

- **UserDefaults**：偏好设置
- **SwiftData**：赛程缓存数据
- **App Groups**：小组件共享数据

### 云端存储

本应用使用 Apple CloudKit 进行数据同步：

- **用途**：同步用户偏好设置到 iCloud
- **存储内容**：通知偏好、收藏列表
- **访问权限**：仅您本人可访问
- **加密**：Apple 标准加密保护

您可以在系统设置中管理 iCloud 同步：

```
设置 → [您的姓名] → iCloud → F1 倒计时
```

---

## 内购信息

### StoreKit 处理

本应用使用 Apple StoreKit 2 处理内购：

- 支付处理由 Apple 安全管理
- 我们**不存储**您的支付信息
- 购买记录仅存储在您的 Apple ID 账户中

### 购买状态

您的 Pro 购买状态存储在：

1. **本地**：UserDefaults（用于快速验证）
2. **Apple 服务器**：通过 Transaction API 验证

---

## 通知

### 本地通知

本应用发送本地通知用于：

- 比赛开始前提醒
- Session 开始前提醒

**说明：**

- 通知完全在本地生成，不经过任何服务器
- 您可以随时在应用设置中关闭通知
- 通知内容仅包含比赛信息，不包含任何个人数据

### 通知权限

首次使用通知功能时，系统会请求您的授权。您可以：

- 允许或拒绝通知权限
- 稍后在系统设置中更改权限

## 第三方服务

### Jolpica-F1 API

本应用使用 Jolpica-F1 公开 API 获取赛程数据：

- **服务提供商**：Jolpica-F1
- **用途**：获取 F1 赛程和比赛信息
- **数据传输**：仅获取公开赛程数据
- **隐私政策**：https://jolpica-f1.github.io/jolpica-f1/

### Apple 服务

本应用使用以下 Apple 服务：

| 服务 | 用途 | 隐私政策 |
|------|------|----------|
| CloudKit | 数据同步 | [Apple 隐私政策](https://www.apple.com/legal/privacy/) |
| StoreKit | 内购处理 | [Apple 隐私政策](https://www.apple.com/legal/privacy/) |
| WidgetKit | 小组件 | 本地运行，无数据传输 |

## 数据安全

### 安全措施

我们采取以下措施保护您的数据：

1. **本地存储**：所有数据仅存储在您的设备
2. **加密传输**：API 请求使用 HTTPS 加密
3. **Apple 安全**：依赖 Apple 的安全框架

### 数据保留

- **本地数据**：保留至您删除应用或清除数据
- **CloudKit 数据**：保留至您删除 iCloud 数据或删除 Apple ID

### 数据删除

您可以通过以下方式删除数据：

1. **删除应用**：删除应用将清除所有本地数据
2. **清除 iCloud 数据**：在 iCloud 设置中删除应用数据
3. **重置偏好**：在应用设置中重置偏好设置
